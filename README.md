# claude-agent-constructor-copilot

A VS Code extension scaffold that opens a Scratch-inspired constructor for agent, skill, and script markdown blocks.

Current flow highlights:
- Left panel palette with drag-and-drop element creation.
- Center workspace with block copy and overuse actions plus usage highlighting.
- Right details popup with layer/color configuration, usage arrows, and file-open links.
- Layer/type multi-select filters at the top of the workspace.
- Workspace markdown decomposition and synchronization back to files in the current workspace.
